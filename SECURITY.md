# Security Policy

## Supported Versions
We currently support the following versions of yTools:

| Version | Supported |
| ------- | --------- |
| 1.2.x   | ✅         |
| < 1.1   | ❌         |

## Reporting a Vulnerability
If you discover a security-related issue (such as a script bug that causes data loss), please do not open a public Issue. Instead, please contact me directly via GitHub or through my professional channels. 

I aim to respond to all security concerns within 48 hours.
